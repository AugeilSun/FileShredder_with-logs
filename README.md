# 文件粉碎机

一个基于 Electron 的安全文件粉碎工具，提供多种安全级别的文件删除功能，确保文件无法被恢复。

## 功能特点

- 🗑️ 多级安全粉碎：提供快速、安全、国防级三种粉碎模式
- 📁 批量处理：支持同时粉碎多个文件和文件夹
- 🔒 安全确认：防止误操作的重要文件粉碎确认机制
- 🎨 现代界面：基于 Electron 的美观用户界面
- 📊 操作日志：记录所有粉碎操作历史
- 📅 多日期日志查看：支持按日期查看历史粉碎记录

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **后端**: Node.js
- **框架**: Electron
- **包管理**: pnpm
- **构建工具**: electron-builder

## 安装与运行

### 环境要求

- Node.js 16+
- pnpm

### 安装依赖

```bash
pnpm install
```

### 开发模式运行

```bash
pnpm start
```

### 构建应用

```bash
pnpm build
```

构建完成后，可执行文件将生成在 `dist-new` 目录中：
- `文件粉碎机 Setup 1.0.0.exe` - 安装包版本
- `文件粉碎机-1.0.0-portable.exe` - 便携版

## 使用说明

1. 启动应用后，点击"添加文件"或"添加文件夹"按钮选择要粉碎的文件
2. 点击"开始粉碎"按钮执行粉碎操作

## 日志查看功能

1. 在应用主界面点击"日志"选项卡
2. 选择日志类型：
   - 应用程序日志：查看应用程序运行日志
   - 粉碎操作日志：查看文件粉碎操作记录
3. 对于粉碎操作日志，可以选择特定日期查看历史记录
4. 点击"刷新"按钮更新日志内容

## 安全说明

- 本工具采用多次覆写文件数据的方式确保文件无法被恢复
- 不同安全级别对应不同的覆写次数和模式
- 粉碎操作不可逆，请谨慎使用
